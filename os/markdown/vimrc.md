# Vim 快捷键速查

> Leader 键：`,`

## 文件与窗口

### 文件操作

| 功能 | 快捷键 |
|--------|--------|
| 强制保存 | `,w` |
| 强制退出 | `qq` |
| 编辑文件 | `,e` |
| 十六进制查看当前文件 | `,x` |

### 分屏

| 功能 | 快捷键 |
|--------|--------|
| 水平分屏 | `,s` |
| 垂直分屏 | `,v` |
| 关闭当前窗口 | `,c` |

### 窗口调整

| 功能 | 快捷键 |
|--------|--------|
| 增大窗口高度 | `++` |
| 缩小窗口高度 | `--` |
| 增大窗口宽度 | `>>` |
| 减小窗口宽度 | `<<` |

### 窗口切换

| 功能 | 快捷键 |
|--------|--------|
| 跳转左窗口 | `Ctrl+h` |
| 跳转下窗口 | `Ctrl+j` |
| 跳转上窗口 | `Ctrl+k` |
| 跳转右窗口 | `Ctrl+l` |

---

## 搜索

| 功能 | 快捷键 |
|--------|--------|
| 向下搜索 | `Space` |
| 向上搜索 | `Ctrl+Space` |
| 清除搜索高亮 | `,<CR>` |

### Visual 模式搜索

| 功能 | 快捷键 |
|--------|--------|
| 向下搜索选中内容 | `*` |
| 向上搜索选中内容 | `#` |
| vimgrep 选中内容 | `gv` |
| vimgrep 模板 | `,g` |

---

## Cscope

| 功能 | 快捷键 |
|--------|--------|
| 查找定义（输入符号） | `,g` |
| 查找光标处定义 | `cg` |
| 查找调用者 | `cc` |
| 查找引用 | `cs` |
| 查找被调用函数 | `cd` |
| 查找字符串 | `ct` |
| 正则搜索 | `ce` |
| 查找文件 | `cf` |
| 查找包含头文件的文件 | `ci` |

---

## QuickFix

| 功能 | 快捷键 |
|--------|--------|
| 打开 QuickFix 窗口 | `,cc` |
| 下一项 | `,n` |
| 上一项 | `,p` |

### Visual 模式

| 功能 | 快捷键 |
|--------|--------|
