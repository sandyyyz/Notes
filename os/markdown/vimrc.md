# Linux 内核源码快速定位

| 功能 | 命令 |
|--------|--------|
| 定位 `schedule()` 定义 | `cscope -dL -1 schedule` |
| 定位 `task_struct` 定义 | `cscope -dL -1 task_struct` |

| 打开定位到的文件 | `vim file_path` |
| 根据 cscope 结果直接打开文件 | `vim $(cscope -dL -1 symbol | awk '{print $1}')` |

# Vim + Cscope 高频工作流

| 步骤 | 命令 |
|--------|--------|
| 查找定义位置 | `cscope -dL -1 symbol` |
| 打开对应文件 | `vim file_path` |
| 继续深入定义 | `Ctrl+]` 或 `cg` |
| 查看引用 | `cs` |
| 查看调用者 | `cc` |
| 查看被调用函数 | `cd` |
| 返回上一层 | `Ctrl+t` |
| 返回上一次跳转位置 | `Ctrl+o` |
| 查看跳转历史 | `:jumps` |
