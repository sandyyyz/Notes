# Vim 快捷键速查

> Leader 键：`,`

---

# 文件与窗口

## 文件操作

| 功能 | 快捷键 |
|--------|--------|
| 强制保存 | `,w` |
| 强制退出 | `qq` |
| 编辑文件 | `,e` |
| 十六进制查看当前文件 | `,x` |

## 分屏

| 功能 | 快捷键 |
|--------|--------|
| 水平分屏 | `,s` |
| 垂直分屏 | `,v` |
| 关闭当前窗口 | `,c` |
| 水平分屏 | `:split` |
| 垂直分屏 | `:vsplit` |
| 平均分配窗口大小 | `Ctrl+w =` |

## 窗口调整

| 功能 | 快捷键 |
|--------|--------|
| 增大窗口高度 | `++` |
| 缩小窗口高度 | `--` |
| 增大窗口宽度 | `>>` |
| 减小窗口宽度 | `<<` |

## 窗口切换

| 功能 | 快捷键 |
|--------|--------|
| 跳转左窗口 | `Ctrl+h` |
| 跳转下窗口 | `Ctrl+j` |
| 跳转上窗口 | `Ctrl+k` |
| 跳转右窗口 | `Ctrl+l` |
